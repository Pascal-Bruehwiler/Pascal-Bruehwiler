Fehlersuche

Aufgabenbeschreibung

In dieser Aufgabe ist es das Ziel, die Fehler in einem Python Programm zu finden und zu verbessern, um danach eine Rätsel zu lösen und das Lösungswort herauszufinden.

Vorgehen

  • Kopiere das Python File mit der Ausgangslage auf dein Computer über den folgenden Link. Fehlersuche.py
  • Öffne das File im Notepad und kopieren den gesamten Inhalt.
  • Öffne den Webcompiler webtigerpython.ethz.ch/
  • Füge den gesamten Inhalt des Fehlersuche.py File auf der Linken Seite der webtigerpython Seite ein.
  • Versuche möglichst alle Fehler im Programm zu beheben.

Weiterführende Aufgabe

Entschlüsselt das angezeigte, verschlüsselte Wort. Der Lernende kann anschliessend das Lösungswort validieren.

Wichtig:
Behaltet das Lösungswort für euch, damit die anderen auch etwas lernen können.
