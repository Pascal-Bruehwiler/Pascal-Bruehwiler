Website Design

Aufgabenbeschreibung

In dieser Aufgabe lernst du, wie du das Aussehen einer Webseite mit Hilfe der Sprache CSS anpassen kannst.

Vorgehen

  • Öffne die zu untersuchende Webseite über den Link. Aufgabe Webseite
  • Wenn die Seite geöffnet ist drücke F12 oder die Tastenkombination Ctrl + Shift + I
  • Klicke im oberen Bereich auf Elements/Elemente. Danach wird dir der eigentliche HTML Code angezeigt.
  • Hier kannst du mittels klicken auf die grauen Pfeile die Struktur der Webseite aufklappen und untersuchen.
  • Wenn du z.B. den Body und den header aufklappst findest du die Zeile Webseite untersuchen mit den vor und nachgestellten 'h1' welchen den Text als Titel definiert.
  • Du findest eine detaillierte Übersicht aller HTML Befehle auf der Webseite w3schools HTML
  • Im rechten Bereich kannst du Styles oben auswählen um die aktuellen Darstellungscodes anzuzeigen. Finde das File Stylesheet.css und klicke darauf. In dieser Datei definierst du das Aussehen der Webseite. z.B. welche Schriftart, Grösse unser Titel 'h1' hat.
  • Du findest eine detaillierte Übersicht aller CSS Befehle auf der Webseite w3schools CSS

Aufgaben

Versuche im Stylesheet.css File folgende Anpassungen zu recherchieren und das File zu verändern. Im linken Bereich verändert sich bei jeder Änderung das Aussehen der Seite.

  1. Der Titel 'Webseite untersuchen' sollte eine Hintergrundfarbe (lightgray) erhalten und Zentriert werden.
  2. Die Gesamte Anleitung sollte eine Abgerundete Umrandung erhalten
  3. Die Codeblocks sollten einen hellgrauen Hintergrund erhalten
  4. Die Beispiele sollten eine Eckige Umrandung erhalten
